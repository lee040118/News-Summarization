# News Summarization chatbot

## 프로젝트 개요

![](/news-summary/chatbot.png)


## 챗봇 기능

1. 서버에서 1시간마다 카테고리 별 네이버 뉴스기사를 크롤링해서 저장한다.
2. 크롤링된 기사들을 각각 요약 모델을 통해 요약문을 생성한다.
3. 카카오톡 챗봇을 통해 유저들이 카테고리 별 뉴스를 요청하면 서버에서 요약문을 전송한다.
